# tp6_quick_template

> tp6 快速开发模板，集成了常用组件

### 开始使用

```bash
git clone git@github.com:feifanshijie/tp6_quick_template.git
cd tp6_quick_template

# 使用composer 安装依赖包
composer install

# 启动内置服务器 
# 访问 http://127.0.0.1:8000
# 输出 Hello, world!
php think run
```

### 目录结构
```
|-app
    |-base          自定义基类
    |-logic         自定义逻辑处理 
    |-validate      自定义验证器
    common.php      公共函数
```
